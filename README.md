On the naming, I agree the current labels could be clearer. We can look to rename items such as "Sponsor Bank" and "Settlement Bank" to more participant focused terminology to reduce confusion. We will review this alongside any other feedback received.
The Participant Output FPS page shows transactions flowing from the Central Infrastructure to direct participants.
On the gateway and Bacs TSP information, I think it would be useful to have a discussion around how changes are communicated so that the underlying data can be kept up to date.
